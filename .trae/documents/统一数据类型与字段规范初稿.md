## 目标更新

* 在统一类型中新增 Project 实体（id、name、createdAt）。

* Generation 与 Preset 等实体以 projectId 关联到 Project，避免重复字段。

## 字段统一规范（更新）

* 尺寸：width、height（所有模型通用）。

* 模型：model（字符串）。

* 分辨率与比例：resolution、aspectRatio（仅 Nano banana 使用）。

* 尺寸来源：sizeFrom（'ratioResolution' | 'custom'）。

* 关联：projectId（字符串，指向 Project.id）。

* URL：outputUrl、coverUrl、previewUrls。

* 用户与时间：userId、createdAt/updatedAt。

## /types/database.ts 内容（拟）

```ts
export type Resolution = '1K' | '2K' | '4K';
export type AspectRatio =
  '1:1' | '2:3' | '3:2' | '3:4' | '4:3' | '4:5' | '5:4' | '9:16' | '16:9' | '21:9';
export type SizeFrom = 'ratioResolution' | 'custom';

export interface Project {
  id: string;
  name: string;
  createdAt: string;
}

export interface GenerationConfig {
  prompt: string;
  width: number;
  height: number;
  model: string;
  lora?: string;
  seed?: number;
  resolution?: Resolution;
  aspectRatio?: AspectRatio;
  sizeFrom?: SizeFrom;
}

export interface Generation {
  id: string;
  userId: string;
  projectId: string;
  outputUrl: string;
  config: GenerationConfig;
  status: 'pending' | 'completed' | 'failed';
  sourceImageUrl?: string;
  createdAt: string;
}

export interface Preset {
  id: string;
  name: string;
  coverUrl: string;
  config: GenerationConfig;
  projectId?: string;
  createdAt: string;
}

export interface Style {
  id: string;
  name: string;
  prompt: string;
  previewUrls: string[];
  updatedAt: string;
}

export interface User {
  id: string;
  displayName: string;
  avatarUrl: string;
  createdAt: string;
}
```

## 使用约束

* Nano banana：按 aspectRatio+resolution 推导 width/height；自定义尺寸时 sizeFrom='custom'。

* 其他模型：仅使用 width/height；aspectRatio/resolution 仅用于 UI 回显。

* Generation、Preset 通过 projectId 关联 Project；缺省时可设为默认项目。
