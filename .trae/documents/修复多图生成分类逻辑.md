## 1. 修正 `isEdit` 判定逻辑
在 `useGenerationService.ts` 的 `handleGenerate` 方法中，移除 `!!editConfig` 作为 `isEdit` 的自动降级逻辑。
```typescript
// 修改前
const isEdit = options.isEdit !== undefined ? options.isEdit : (configOverride?.isEdit || freshConfig.isEdit || !!editConfig);

// 修改后
const isEdit = options.isEdit !== undefined ? options.isEdit : (configOverride?.isEdit || freshConfig.isEdit || false);
```

## 2. 编辑器关闭时重置状态
在 `playground.tsx` 中，更新关闭编辑器的逻辑，确保重置全局 `config`。
```typescript
const handleCloseEditor = () => {
  setIsEditorOpen(false);
  updateConfig({ isEdit: false, editConfig: undefined, parentId: undefined });
};
```

## 3. 编辑保存时显式标记
在 `playground.tsx` 的 `handleSaveEditedImage` 中，调用 `handleGenerate` 时显式传入 `isEdit: true`。
```typescript
if (shouldGenerate) {
  handleGenerate({ editConfig, isEdit: true });
}
// 保存后也需要重置全局编辑状态
updateConfig({ isEdit: false, editConfig: undefined });
```
